# Student Grading App

Application C# réalisée avec Visual Studio 2022 pour automatiser le calcul des notes de chaque étudiant d’une classe.

## Fonctionnalités

	- Calcul de moyennes;
	- Affichage es résultats;

## Technologies

	- C#;
	- Visual Studio 2022;
	- .NET;

## Lancer le projet

1. Cloner le projet depuis le dépôt GitHub;
2. Ouvrir le projet dans Visual Studio 2022;
3. Compiler le projet;
4. Exécuter le projet.
